# mydsaa
数据结构和算法(Data Structure and Algorithm)的简单学习

## 数据结构
数据结构：计算机存储和组织数据的方式，是相互之间存在一种或多种特定关系的元素的集合。

数据结构的基本功能：

- 插入一条数据
- 删除一条数据
- 修改一条数据
- 查询一条数据 
- 遍历

常用的数据结构：

- 数组Array
- 栈Stack
- 队列Queue
- 链表LinkedList
- 树Tree
- 哈希表Hash
- 堆Heap
- 图Graph

常用数据结构的优缺点

|数据结构|优点|缺点|
|---|---|---|
|无序数组|插入快O(1)、按下标修改和查询快O(1)|按值删除、修改和查询慢O(n)；元素类型要求一致；大小固定|

## 算法
算法：简单来说就是解决问题的步骤。

算法的五个特性：

- 有穷性
- 确定性
- 可行性
- 有输入
- 有输出

算法的设计原则

- 正确性
- 可读性
- 健壮性
- 高效低存储
