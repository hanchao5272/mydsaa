## 链表
链表：由一连串节点构成的线性结构，每个节点包含节点数据(data)以及一个或两个指向上一个/下一个节点的引用。

链表优缺点：

- 大小不固定，可以克服数组使用时需要事先知道数据大小的缺点
- 失去了通过下标随机读取的优点
- 因为要存储引用，需要更多的存储空间

## 单向链表
单向链表：

1. 节点构成：data + next
2. 插入节点：new.next = head
3. 删除头节点：head = head.next
4. 删除节点：some.next = some.next.next

算法复杂度：

- 插入/删除头节点：O(1)
- 查询/修改/删除指定节点：O(n)
